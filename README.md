<h1>Apexx jQuery Carousel</h1>
<p>An easy-to-install jQuery powered responsive carousel plugin.</p>
<h2>Features</h2>
<ul>
  <li>Responsive</li>
  <li>Powered by jQuery</li>
  <li>Accepts HTML contents</li> 
  <li>Lightweight</li>
</ul>
